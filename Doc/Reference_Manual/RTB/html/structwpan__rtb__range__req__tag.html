uint8_t wpan_rtb_range_req_tag::CoordinatorAddrMode

The Coordinator addressing mode for the received range request. This value can take one of the following values:

0x00 = no Coordinator address; this is a regular range request

0x02 = 16 bit short address; 0x03 = 64 bit extended address; This is a remote range request
